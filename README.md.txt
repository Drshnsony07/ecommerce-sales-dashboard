# E-Commerce Sales Dashboard

## Project Overview
This project analyzes e-commerce sales data using Python and Power BI to identify sales trends, top products, and business insights.

## Tools Used
- Python (Pandas, NumPy)
- Power BI
- SQL
- Excel

## Dashboard Features
- Total Sales
- Total Orders
- Total Profit
- Monthly Sales Trend
- Top 10 Products
- Sales by Category
- Interactive Filters

## Key Insights
- Technology category generated the highest sales.
- West region contributed the most revenue.
- November had the highest sales.
- Copiers were the most profitable products.
